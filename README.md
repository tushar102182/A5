1️⃣ What is the difference between var, let, and const?
Answer:
var:Function scoped,Can redeclare,Can update.
let:Block scoped,Can update and Cannot redeclare.
const :It is also a Block scoped,Cannot update and Cannot redeclare.

2️⃣ What is the spread operator (...)?
Answer: Spread expands elements from array or object. example :  const user = {name:"Tushar"};
const updated = {...user, age:20};

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer : map() creates data after operation . filter() only gets the values that fulfill the condition. forEach() donot return anything it only runs like a loop.
